# coolplanet-challenge

To run the frontend project: 

  - cd to `frontend`
  - run 'npm install'
  - run 'npm start'
  
  
 To run the backend project locally:

  - cd to `node`
  - run 'npm install'
  - run 'npm run dev'
  - if you need to restart the node project, in your terminal type 'rs' and press enter
